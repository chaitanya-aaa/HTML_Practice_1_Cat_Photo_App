# 🐱 Cat Photo App

## 📌 Project Overview

This project demonstrates the creation of a simple **HTML-based Cat Photo App webpage**.  
The webpage displays cat images, provides links to external cat photo galleries, and allows users to submit information about their cat through a form.

The main objective of this project is to practice **basic HTML structure, images, lists, hyperlinks, and form elements**.

This project highlights fundamental web development concepts including **HTML page structure, image embedding, lists, hyperlinks, and form inputs**.

---

## 🎯 Problem Statement

To design a simple webpage that displays cat photos, shows lists of things cats love and hate, and allows users to submit their cat information using an HTML form.

---

## 🛠️ Project Modules Covered

### 1️⃣ HTML Document Structure
- Created the base HTML document using `html`, `head`, and `body` tags.
- Added metadata such as charset and viewport.

### 2️⃣ Headings & Text Content
- Used heading tags (`h1`, `h2`, `h3`) to structure the webpage.
- Displayed descriptive text using paragraph tags.

### 3️⃣ Images & Hyperlinks
- Added cat images using the `<img>` tag.
- Used anchor tags `<a>` to link external cat photo websites.

### 4️⃣ Lists
- Used **unordered lists** to display things cats love.
- Used **ordered lists** to display things cats hate.

### 5️⃣ Figure & Caption
- Displayed images with captions using `<figure>` and `<figcaption>`.

### 6️⃣ Form Implementation
- Created a form to collect cat details.
- Used:
  - Radio buttons
  - Checkboxes
  - Text input field
  - Submit button

### 7️⃣ Footer Section
- Added footer containing contact details and copyright information.
